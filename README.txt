# Turing_Machine

Considero que el proyecto es robusto porque verifica las cadenas y condiciones en la
máquina de Turing para evaluar casos como el 0 en las operaciones unitarias. 
También porque cuenta correctamente los pasos y señala en qué estado está la máquina
en cada una de las transiciones. 

El funcionamiento del programa es sencillo, en un ComboBox se escoge la operación a realizar
y luego se ingresa la cadena. Al iniciar se desplegará la cinta y se mostrará un mensaje
cuando la cadena termine de ejecutarse. Para reiniciar la máquina se deberá pulsar el botón
REINICIAR.
